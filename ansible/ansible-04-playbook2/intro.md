# Ansible Playbook. Часть 2 — Переменные, Jinja2, условия

В первой части познакомились с базовыми `vars` и простым `when`.
Теперь идём глубже: вся иерархия переменных, мощные Jinja2-фильтры,
сложные шаблоны конфигов и умные условия с `register`.

```
Что нового:
  vars_files       <- переменные из YAML-файлов
  vars_prompt      <- запрос у пользователя при запуске
  register         <- захватить вывод задачи в переменную
  hostvars/groups  <- магические переменные Ansible
  Jinja2 фильтры   <- | default | upper | join | selectattr ...
  {% for %}        <- циклы в шаблонах
  failed_when      <- переопределить что считать ошибкой
  changed_when     <- переопределить что считать изменением
```
