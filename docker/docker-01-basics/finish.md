## Урок 1 пройден!

Шпаргалка:
- `docker run -d --name имя -p хост:контейнер образ` — запустить в фоне
- `docker ps` / `docker ps -a` — список запущенных / всех
- `docker logs имя` — логи контейнера
- `docker exec -it имя bash` — зайти внутрь
- `docker stop имя` — остановить
- `docker rm имя` — удалить
- `docker run --rm образ` — запустить и авто-удалить после

В следующем уроке разберём образы и Docker Hub.
