# Поздравляем! 🎉

Вы успешно завершили модуль **«Docker: Первые шаги»**!

## Что вы освоили

- **`docker run`** — запуск контейнеров, флаги `-d`, `-p`, `--name`, `--rm`
- **`docker ps`** — просмотр контейнеров, флаги `-a`, `-q`, `--filter`
- **`docker stop`** — плавная остановка через SIGTERM, флаг `-t`
- **`docker rm`** — удаление контейнеров, флаг `-f`, `docker container prune`

## Шпаргалка

| Команда | Описание |
|---------|----------|
| `docker run -d IMAGE` | Запуск в фоне |
| `docker run --rm IMAGE` | Авто-удаление после завершения |
| `docker run --name NAME IMAGE` | Задать имя |
| `docker run -p HOST:CONT IMAGE` | Проброс порта |
| `docker ps` | Запущенные контейнеры |
| `docker ps -a` | Все контейнеры |
| `docker stop NAME` | Остановить контейнер |
| `docker rm NAME` | Удалить контейнер |
| `docker rm -f NAME` | Принудительно удалить |
