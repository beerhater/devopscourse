# Шаг 1: Первый docker run

`docker run` — главная команда для запуска контейнеров. Когда вы выполняете `docker run <image>`, Docker:
1. Ищет образ локально
2. Если не находит — скачивает его из Docker Hub
3. Создаёт контейнер на основе образа и запускает его

## Задание

Запустите свой первый контейнер:

```bash
docker run hello-world
```{{execute}}

Вы увидите сообщение `Hello from Docker!` — это значит, что контейнер успешно запустился, выполнил задачу и автоматически завершился.

> **Важно:** контейнер на основе `hello-world` завершается сразу после вывода сообщения. Это нормально — не все контейнеры работают постоянно.
