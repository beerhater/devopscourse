# Шаг 2: Фоновый режим и проброс портов

Контейнеры можно запускать в разных режимах:

- **`-d`** (detached) — фоновый режим, контейнер работает независимо от терминала
- **`-p HOST:CONTAINER`** — проброс порта с хоста в контейнер
- **`--name`** — задать имя контейнеру

## Задание

Запустите `nginx` в фоновом режиме с именем и пробросом порта:

```bash
docker run -d -p 8080:80 --name my-nginx nginx
```{{execute}}

Docker выведет длинный ID контейнера — это его уникальный идентификатор. Проверьте, что nginx отвечает:

```bash
curl http://localhost:8080
```{{execute}}
