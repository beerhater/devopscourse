## Первый контейнер

Команда `docker run hello-world` — стандартный способ проверить
что Docker работает. Образ `hello-world` весит несколько килобайт
и просто выводит сообщение.

---

1. Запустите тестовый контейнер:
`docker run hello-world`

Прочитайте вывод — Docker сам объясняет что только что произошло!

2. Посмотрите список **всех** контейнеров (включая завершённые, флаг `-a`):
`docker ps -a`

Вы увидите контейнер со статусом `Exited` — он отработал и завершился.

3. Обратите внимание на колонки:
- `CONTAINER ID` — уникальный ID
- `IMAGE` — из какого образа создан
- `STATUS` — текущее состояние
- `NAMES` — Docker сам придумал случайное имя (например `funny_darwin`)
