## Фоновый запуск: флаг -d

По умолчанию контейнер "занимает" терминал: вы видите его вывод,
но не можете ввести другую команду.

Флаг `-d` (detached) запускает контейнер в **фоне**.
Флаг `--name` даёт контейнеру **понятное имя** вместо случайного.
Флаг `-p` проброс порта: `-p 8080:80` означает "порт 8080 на хосте → порт 80 в контейнере".

---

1. Запустите nginx в фоне:
`docker run -d --name web -p 8080:80 nginx`

2. Убедитесь, что контейнер работает:
`docker ps`

Статус должен быть `Up`.

3. Проверьте что nginx отвечает:
`curl -s http://localhost:8080 | head -5`

Вы увидите HTML-код страницы nginx.

4. Посмотрите логи контейнера:
`docker logs web`
