# Шаг 3: Просмотр контейнеров — docker ps

`docker ps` показывает список **запущенных** контейнеров.

## Основные варианты команды

Посмотрите запущенные контейнеры:

```bash
docker ps
```{{execute}}

Флаг `-a` показывает **все** контейнеры — и работающие, и остановленные:

```bash
docker ps -a
```{{execute}}

Флаг `-q` выводит только ID (удобно для скриптов):

```bash
docker ps -q
```{{execute}}

## Задание

Убедитесь, что `my-nginx` присутствует в списке запущенных и запомните его CONTAINER ID:

```bash
docker ps --filter "name=my-nginx"
```{{execute}}
