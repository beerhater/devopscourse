# Поздравляем! 🎉

Вы успешно завершили модуль **«Docker: Образы»**!

## Что вы освоили

- **`docker pull`** — скачивание образов из Docker Hub
- **`docker images`** — просмотр локальных образов, флаги `-q`, `--format`, фильтрация
- **Теги** — версионирование образов, важность конкретных тегов вместо `latest`
- **`docker search`** — поиск образов, фильтрация по `is-official` и `stars`
- **`docker rmi`** — удаление образов, `docker image prune -a`
- **`docker inspect`** — подробная информация об образе

## Шпаргалка

| Команда | Описание |
|---------|----------|
| `docker pull IMAGE:TAG` | Скачать образ |
| `docker images` | Список всех образов |
| `docker images -q` | Только ID образов |
| `docker images NAME` | Фильтр по имени |
| `docker search NAME` | Поиск на Docker Hub |
| `docker inspect IMAGE` | Подробная информация |
| `docker rmi IMAGE` | Удалить образ |
| `docker image prune -a` | Удалить все неиспользуемые |
| `docker system df` | Использование диска |
