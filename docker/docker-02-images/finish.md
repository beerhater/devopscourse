## Урок 2 пройден!

Шпаргалка:
- `docker search имя` — поиск на Docker Hub
- `docker pull имя:тег` — скачать образ
- `docker images` — список локальных образов
- `docker inspect имя` — полные метаданные
- `docker history имя` — слои образа
- `docker rmi имя:тег` — удалить образ
- `docker image prune -a` — удалить все неиспользуемые

Дальше — создаём свои образы через Dockerfile!
