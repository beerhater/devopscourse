# Docker: Образы

В предыдущем модуле вы запускали контейнеры, используя готовые образы. Теперь разберёмся, **что такое образ**, как им управлять и где их искать.

В этом модуле вы освоите:

- **`docker pull`** — скачивание образов из реестра
- **`docker images`** — просмотр локальных образов
- **`docker rmi`** — удаление образов
- **Docker Hub** — публичный реестр образов
- **Теги** — версионирование образов

> **Среда:** Docker уже установлен. Проверьте: `docker version`{{execute}}
