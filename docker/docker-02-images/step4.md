## Инспекция образа

`docker inspect` показывает полные метаданные образа в JSON:
переменные окружения, команду запуска, открытые порты, метки.

---

1. Инспектируйте образ nginx:
`docker inspect nginx`

2. Найдите только команду запуска по умолчанию:
`docker inspect nginx --format='{{.Config.Cmd}}'`

3. Посмотрите переменные окружения:
`docker inspect nginx --format='{{.Config.Env}}'`

4. Посмотрите историю слоёв образа:
`docker history nginx`

Каждая строка — это один слой. Видите как строился образ шаг за шагом?
