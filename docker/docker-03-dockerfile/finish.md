## Урок 3 пройден!

Вы знаете все ключевые инструкции Dockerfile:
- `FROM` — базовый образ (всегда первая строка)
- `RUN` — выполнить при сборке (объединяйте через &&)
- `COPY` — скопировать файлы
- `WORKDIR` — рабочая директория
- `ENV` — переменные окружения
- `EXPOSE` — документировать порт
- `CMD` — команда по умолчанию
