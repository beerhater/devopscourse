# Docker: Написание Dockerfile

До сих пор вы использовали **чужие образы** с Docker Hub. Настало время создавать **собственные**!

**Dockerfile** — это текстовый файл с набором инструкций, по которым Docker собирает образ. Каждая инструкция создаёт новый слой.

В этом модуле вы освоите ключевые инструкции:

- **`FROM`** — базовый образ
- **`RUN`** — выполнение команд при сборке
- **`COPY`** — копирование файлов в образ
- **`CMD`** — команда по умолчанию при запуске
- **`WORKDIR`** — рабочая директория
- **`ENV`** — переменные окружения
- **`EXPOSE`** — документирование портов
- **`ARG`** — аргументы сборки

> Проверьте среду: `docker version`{{execute}}
