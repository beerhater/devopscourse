## Урок 3: Написание Dockerfile

Dockerfile — это текстовый файл с инструкциями по сборке образа.
Каждая инструкция создаёт новый **слой** в образе.
Слои кешируются: если вы изменили только последние строки,
Docker не будет пересобирать первые.

---

### Все инструкции Dockerfile:

| Инструкция  | Что делает                                        |
|-------------|--------------------------------------------------|
| `FROM`      | Базовый образ                                    |
| `RUN`       | Выполнить команду при сборке                     |
| `COPY`      | Скопировать файлы с хоста в образ                |
| `ADD`       | Как COPY, но умеет распаковывать архивы          |
| `ENV`       | Переменная окружения                             |
| `EXPOSE`    | Документировать порт (не открывает реально)      |
| `CMD`       | Команда по умолчанию (можно переопределить)      |
| `ENTRYPOINT`| Точка входа (сложнее переопределить)             |
| `WORKDIR`   | Рабочая директория внутри образа                 |
| `USER`      | От какого пользователя выполнять команды         |
