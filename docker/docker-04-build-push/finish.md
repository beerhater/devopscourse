# Поздравляем! 🎉

Вы успешно завершили модуль **«Docker: Сборка и публикация образов»**!

## Что вы освоили

- **`docker build`** — флаги `-t`, `-f`, `--no-cache`, `--build-arg`, контекст сборки
- **Кеш слоёв** — принцип работы, правило "нестабильные слои вниз"
- **`docker tag`** — присвоение нескольких тегов одному образу
- **Multi-stage builds** — разделение сборки и финального образа
- **`docker login`** — авторизация в Docker Hub и других реестрах
- **`docker push`** — публикация образов, отправка только новых слоёв
- **Локальный реестр** — `registry:2` для приватного хранения образов

## Шпаргалка

| Команда | Описание |
|---------|----------|
| `docker build -t name:tag .` | Собрать образ |
| `docker build --no-cache .` | Сборка без кеша |
| `docker build --build-arg K=V .` | Передать ARG |
| `docker build --target stage .` | Собрать до этапа |
| `docker tag src:tag dst:tag` | Добавить тег |
| `docker login` | Авторизация |
| `docker push name:tag` | Опубликовать образ |
| `docker history image` | Слои образа |
