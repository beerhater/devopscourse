# Docker: Сборка и публикация образов

Вы умеете писать Dockerfile. Теперь разберёмся, как собирать образы профессионально и делиться ими с командой через реестры.

В этом модуле вы освоите:

- **`docker build`** — флаги, контекст сборки, кеш слоёв
- **`docker tag`** — присвоение тегов образам
- **Multi-stage builds** — уменьшение финального образа
- **`docker login`** — авторизация в реестре
- **`docker push`** — публикация образа на Docker Hub

> Проверьте среду: `docker version`{{execute}}
