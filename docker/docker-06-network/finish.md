## Урок 6 пройден!

Шпаргалка:
- `docker network ls` — список сетей
- `docker network create имя` — создать сеть
- `docker network inspect имя` — детали
- `docker network connect сеть контейнер` — добавить контейнер в сеть
- `docker network rm имя` — удалить сеть
- `--network имя` — указать сеть при запуске

Главное правило: создавайте пользовательскую bridge-сеть —
контейнеры в ней находят друг друга по имени.
