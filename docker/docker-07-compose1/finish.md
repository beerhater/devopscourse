## Урок 7 пройден!

Шпаргалка:
- `docker compose up -d` — запустить стек в фоне
- `docker compose ps` — статус сервисов
- `docker compose logs -f сервис` — логи в реальном времени
- `docker compose stop/start` — остановить/запустить без удаления
- `docker compose down` — остановить и удалить (данные живы)
- `docker compose down -v` — удалить вместе с томами
