## Урок 7: Docker Compose (Часть 1)

Реальное приложение состоит из нескольких сервисов:
веб-сервер + база данных + кэш (Redis) + очередь сообщений.

Без Compose вам нужно вручную:
1. Создать сеть
2. Запустить postgres с нужными переменными
3. Запустить redis
4. Запустить backend с правильными параметрами
5. Запустить nginx как прокси

Docker Compose описывает весь этот стек в **одном YAML-файле**
и запускает его одной командой `docker compose up`.

---

### Версии:
- `docker-compose` (v1) — устаревшая утилита, отдельная установка
- `docker compose` (v2) — встроен в Docker, современный стандарт

В этом курсе используем `docker compose` (v2).
