## Урок 8 пройден! Блок Docker завершён!

Финальная шпаргалка по Docker Compose:
- `depends_on: service: condition: service_healthy` — ждать готовности
- `healthcheck` — описать как проверять здоровье сервиса
- `.env` файл — выносить секреты из yaml
- `${VAR}` — подставлять переменные из .env
- `docker compose up -d --scale app=3` — масштабировать сервис
- `networks:` — изолировать сервисы по сегментам сети
