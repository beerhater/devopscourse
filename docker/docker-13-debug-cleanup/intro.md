## Docker: Debug и cleanup

Со временем на host накапливаются:

- остановленные контейнеры;
- dangling images;
- забытые временные артефакты.

В этом уроке вы сделаете маленький audit и безопасный cleanup.

В уроке акцент сделан на безопасной уборке.
Мы сначала собираем картину по системе, потом находим конкретные лишние объекты и только потом удаляем их по одному.
Такой подход намного полезнее, чем привычка запускать агрессивные prune-команды без понимания последствий.

Этот урок хорошо готовит к работе на shared Docker host, где careless cleanup может затронуть чужие контейнеры и данные.
