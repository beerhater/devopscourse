# Pod — минимальная единица K8s

**Pod** — обёртка над одним (иногда несколькими) контейнерами, которые:
- Всегда работают на одном узле
- Делят один IP-адрес
- Делят общие тома (volumes)

В 95% случаев в поде живёт ровно один контейнер. K8s управляет не контейнерами — только Подами.

> Проверим кластер: `kubectl get nodes`{{execute}}
