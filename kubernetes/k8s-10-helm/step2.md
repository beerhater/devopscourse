```{{execute}}

```bash

```{{execute}}

## What each file does

```bash
cat << 'EOF'
myapp-chart/
  Chart.yaml          <- chart metadata (name, version, description)
  values.yaml         <- default values for templates
  charts/             <- sub-charts (dependencies)
  templates/
    deployment.yaml   <- K8s Deployment template
    service.yaml      <- K8s Service template
    serviceaccount.yaml
    ingress.yaml      <- K8s Ingress template
    hpa.yaml          <- HorizontalPodAutoscaler
    _helpers.tpl      <- named templates (reusable snippets)
    NOTES.txt         <- post-install notes shown to user
    tests/
      test-connection.yaml
EOF
```{{execute}}

```bash
# Default values.yaml generated by helm create
