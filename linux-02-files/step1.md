## Читаем файл

Команда `cat` выводит содержимое файла прямо в терминал.

Для начала создадим файл с текстом:

`echo "Hello DevOps" > hello.txt`

Команда `echo` выводит текст, а `>` перенаправляет его в файл.

**Задание:**
1. Создайте файл: `echo "Hello DevOps" > hello.txt`
2. Прочитайте его: `cat hello.txt`

Вы должны увидеть текст `Hello DevOps`.
