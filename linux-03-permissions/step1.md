## Читаем права доступа

Команда `ls -la` показывает подробный список файлов с правами.

Создайте файл и посмотрите на его права:

`touch myfile.txt`
`ls -la myfile.txt`

Вы увидите что-то вроде:

`-rw-r--r-- 1 root root 0 Mar 16 12:00 myfile.txt`

Разбираем строку слева направо:
- `-` — это обычный файл (d — директория)
- `rw-` — владелец может **читать** и **писать**
- `r--` — группа может только **читать**
- `r--` — все остальные могут только **читать**

**Задание:**
1. Создайте файл: `touch myfile.txt`
2. Посмотрите права: `ls -la myfile.txt`
