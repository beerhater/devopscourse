## Меняем права через chmod

`chmod` меняет права доступа к файлу.
Права задаются цифрами:
- `4` — чтение (r)
- `2` — запись (w)
- `1` — выполнение (x)

Складываем цифры для каждой группы: **владелец / группа / остальные**

Примеры:
- `chmod 644 файл` → владелец читает+пишет, остальные только читают
- `chmod 600 файл` → только владелец читает+пишет, остальные — ничего
- `chmod 777 файл` → все могут всё (используйте осторожно!)

**Задание:**
Сделайте файл доступным только владельцу:

`chmod 600 myfile.txt`

Проверьте: `ls -la myfile.txt`
Должны увидеть: `-rw-------`
