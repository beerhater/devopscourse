## Смотрим открытые порты через ss

`ss` (Socket Statistics) — современная замена `netstat`.
Показывает открытые порты и активные соединения.

Самая полезная комбинация флагов — `ss -tulnp`:
- `-t` — TCP соединения
- `-u` — UDP соединения
- `-l` — только слушающие порты (listening)
- `-n` — показывать числа вместо имён сервисов
- `-p` — показывать процесс который держит порт

**Задание:**
1. Установите и запустите nginx:

`apt-get install -y nginx`
`systemctl start nginx`

2. Посмотрите открытые порты:

`ss -tulnp`

Найдите в выводе строку с портом `80` — это nginx.

3. Сохраните вывод в файл:

`ss -tulnp > ports.txt`
