## Базовый iptables

`iptables` — файрвол Linux. Управляет тем какой трафик
разрешён, а какой блокируется.

Основные цепочки (chains):
- `INPUT` — входящий трафик к серверу
- `OUTPUT` — исходящий трафик от сервера
- `FORWARD` — транзитный трафик

Основные действия (targets):
- `ACCEPT` — разрешить
- `DROP` — заблокировать молча
- `REJECT` — заблокировать с уведомлением

**Задание:**
1. Посмотрите текущие правила:

`iptables -L -n`

2. Разрешите входящий трафик на порт 8080:

`iptables -A INPUT -p tcp --dport 8080 -j ACCEPT`

3. Проверьте что правило добавилось:

`iptables -L INPUT -n`

4. Сохраните правила в файл:

`iptables -L -n > iptables_rules.txt`
