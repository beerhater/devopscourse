## Перемещаем и переименовываем

Команда `mv` (Move) делает две вещи:
- Перемещает файл в другую папку
- Переименовывает файл (это просто перемещение в ту же папку с новым именем)

**Задание:**
1. Создайте папку для архива: `mkdir archive`
2. Переместите backup.txt туда: `mv backup.txt archive/backup.txt`
3. Проверьте: `ls archive/`
