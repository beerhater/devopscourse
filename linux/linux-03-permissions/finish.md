## 🎉 Урок 3 пройден!

Теперь вы понимаете одну из фундаментальных концепций Linux:

- `ls -la` — видим права в формате `rwxrwxrwx`
- `chmod 600` — закрываем файл только для владельца
- `chmod +x` — делаем скрипт исполняемым
- `chown user:group` — передаём владение файлом
- `chgrp группа файл` — меняем группу без смены владельца
- `umask 027` — делаем новые файлы безопаснее по умолчанию
- `chmod 1777 директория` — включаем sticky bit для общей папки
