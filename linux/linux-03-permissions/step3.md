## Исполняемый файл

В Linux чтобы запустить скрипт — ему нужно дать право на выполнение.
Именно поэтому мы всегда делаем `chmod +x` для bash-скриптов.

`+x` означает "добавить право на выполнение" для всех.

**Задание:**
1. Создайте простой скрипт:

`echo '#!/bin/bash' > myscript.sh`
`echo 'echo "Скрипт работает!"' >> myscript.sh`

2. Попробуйте запустить без прав: `./myscript.sh` (получите ошибку Permission denied)
3. Дайте право на выполнение: `chmod +x myscript.sh`
4. Запустите снова: `./myscript.sh`
