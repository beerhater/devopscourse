## Sticky bit для общей директории

Sticky bit нужен для общих директорий, куда писать могут многие пользователи,
но удалять чужие файлы нельзя.

Самый известный пример — `/tmp`.
У него права вида `drwxrwxrwt`: буква `t` в конце и есть sticky bit.

**Задание:**
1. Создайте общую директорию: `mkdir -p /shared-dropbox`
2. Дайте всем право писать в неё, но с sticky bit: `chmod 1777 /shared-dropbox`
3. Создайте тестовый файл: `touch /shared-dropbox/readme.txt`
4. Проверьте права: `ls -ld /shared-dropbox`

Вы должны увидеть права вида `drwxrwxrwt`.
Такой подход часто используют для временных файлов и shared drop-зон.
