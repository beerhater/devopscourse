## Поиск процесса через pgrep и pkill

Искать PID вручную через `ps aux | grep` неудобно.
Для этого есть специальные команды:

- `pgrep имя` — находит PID процессов по имени
- `pkill имя` — убивает все процессы с таким именем сразу

**Задание:**
1. Запустите несколько фоновых процессов:

`sleep 3000 &`
`sleep 4000 &`
`sleep 5000 &`

2. Найдите все PID через pgrep:

`pgrep sleep`

Увидите несколько PID — по одному на каждый sleep.

3. Убейте все sleep одной командой:

`pkill sleep`

4. Убедитесь что все процессы исчезли:

`pgrep sleep`

Вывод должен быть пустым — процессов не осталось.
