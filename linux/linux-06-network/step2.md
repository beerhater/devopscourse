## Делаем HTTP-запросы через curl

`curl` — универсальный инструмент для работы с HTTP.
В DevOps он используется постоянно: проверить API, скачать файл,
отправить данные на сервер.

Полезные флаги:
- `-I` — показать только заголовки ответа (HEAD-запрос)
- `-o файл` — сохранить ответ в файл
- `-s` — тихий режим без прогресс-бара
- `-w "%{http_code}"` — вывести только HTTP код ответа

**Задание:**
1. Сделайте GET-запрос и посмотрите ответ:

`curl https://httpbin.org/get`

2. Посмотрите только HTTP-заголовки:

`curl -I https://httpbin.org/get`

3. Проверьте HTTP-код ответа (должен быть 200):

`curl -s -o /dev/null -w "%{http_code}" https://httpbin.org/get > curl_code.txt`

`cat curl_code.txt`
