## Анализируем соединения через netstat

`netstat` — классический инструмент анализа сети.
Хотя `ss` быстрее, `netstat` до сих пор встречается на многих серверах.

Полезные флаги `netstat -tulnp`:
- `-t` — TCP
- `-u` — UDP
- `-l` — слушающие порты
- `-n` — числовой вывод
- `-p` — имя процесса

**Задание:**
1. Установите net-tools (пакет с netstat):

`apt-get install -y net-tools`

2. Посмотрите все активные соединения:

`netstat -tulnp`

3. Найдите только порт 80:

`netstat -tulnp | grep :80`

4. Посчитайте количество активных TCP соединений:

`netstat -tn | grep ESTABLISHED | wc -l`

Команда `wc -l` считает количество строк.
