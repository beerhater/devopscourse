## 🎉 Урок 7 пройден!

Теперь вы владеете пакетным менеджером на уровне уверенного сисадмина:

- `apt update` — узнать о новых версиях
- `apt install -y pkg` — установить без вопросов (идеально для скриптов)
- `apt search/show` — найти информацию
- `apt upgrade -s` — проверить, что обновится
- `apt purge -y pkg` — удалить без следов
- `apt autoremove -y` — очистить систему от брошенных зависимостей
- `apt clean` — почистить кэш на диске

В следующих двух уроках начинается самое интересное —
мы начнём писать **Bash-скрипты**, чтобы автоматизировать рутину!
