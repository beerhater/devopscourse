## Проверка установленного

Иногда нужно понять, установлена ли уже программа на сервере.
Для этого можно использовать:
- `which программа` (показывает путь к исполняемому файлу)
- `apt list --installed` (список всех установленных пакетов)

**Задание:**
1. Найдите, где физически лежит программа jq:

`which jq > jq_path.txt`

2. Проверьте статус пакета tree через apt list:

`apt list --installed | grep tree > tree_status.txt`

3. Посмотрите результаты:
`cat jq_path.txt`
`cat tree_status.txt`
