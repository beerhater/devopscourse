## Урок 8 пройден!

Теперь вы знаете фундамент Bash:
- `#!/bin/bash` — shebang
- `VAR="value"` — переменные (без пробелов!)
- `$(команда)` — результат команды в переменную
- `$(( 2 + 2 ))` — математика
- `if [ ... ]; then ... fi` — условия
- `for i in ...; do ... done` — циклы
