## Пишем первую функцию

Допустим, ваш скрипт 5 раз выводит текущее время и статус сервера.
Без функций вам пришлось бы писать одни и те же строки 5 раз.
Функция решает это: описываете логику один раз, вызываете по имени.

**Синтаксис функции:**
```
function имя_функции() {
    # тело функции
    команды
}

# Вызов — просто пишете имя как команду:
имя_функции
```

Важно: функция должна быть **объявлена раньше** чем вы её вызываете.

---

**Шаг 1.** Создайте скрипт:
`nano func.sh`

**Шаг 2.** Введите код:
```
#!/bin/bash

function show_status() {
    echo "----------------------------"
    echo "Время: $(date '+%H:%M:%S')"
    echo "Сервер работает нормально"
    echo "----------------------------"
}

echo "Первая проверка:"
show_status

echo "Вторая проверка:"
show_status
```
Сохраните: `Ctrl+O` → Enter → `Ctrl+X`

**Шаг 3.** Запустите:
`chmod +x func.sh && ./func.sh`

Обратите внимание: код внутри функции написан один раз, но выполнился дважды.
