## Урок 11 пройден!

Теперь у вас есть практический набор инструментов для чтения логов и конфигов:

- `head -n N файл` — посмотреть начало файла
- `tail -n N файл` — посмотреть конец файла
- `less файл` — интерактивно читать большой файл
- `grep строка файл` — искать точное совпадение
- `grep -i строка файл` — искать без учёта регистра
- `grep -r строка каталог` — искать по дереву каталогов
- `find путь -type f -name "*.conf"` — искать файлы по имени
- `wc -l файл` — считать строки
- `sort | uniq -c` — группировать и считать повторы

В следующем уроке соберём всё это в цепочки через pipes и научимся управлять stdout/stderr.
