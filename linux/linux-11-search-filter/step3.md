## grep -r по дереву каталогов

Когда конфигов много, искать в каждом файле руками бессмысленно.
Используйте `grep -r` — рекурсивный поиск по каталогу.

```bash
grep -r "DB_HOST" /opt/search-lab/configs > /root/db_host_hits.txt
cat /root/db_host_hits.txt
```{{execute}}

Так вы сразу увидите:
- в каких файлах встречается ключ;
- где искать переменные подключения;
- не дублируется ли настройка в нескольких местах.

Это типичная задача при разборе чужого проекта или старого сервера.
