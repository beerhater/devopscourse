## Урок 12 пройден! Linux-блок завершён

Теперь у вас есть сильная практическая база по Linux для DevOps:

- `|` — передать вывод одной команды в другую
- `>` — перезаписать файл выводом команды
- `>>` — дописать вывод в конец файла
- `<` — подать файл на stdin команды
- `2>` — сохранить ошибки отдельно
- `2>&1` — объединить stdout и stderr

Самое важное: вы научились не просто запускать команды по одной,
а собирать из них полезные цепочки для диагностики и автоматизации.
