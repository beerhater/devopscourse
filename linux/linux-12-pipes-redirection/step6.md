## Pipes — передаём вывод между командами

Символ `|` передаёт вывод одной команды на вход другой.

Это значит:
- первая команда находит данные;
- следующая команда работает уже только с нужным подмножеством.

```bash
grep -i "error" /opt/pipes-lab/events.log | tail -n 2 > /root/recent_errors.txt
cat /root/recent_errors.txt
```{{execute}}

Здесь цепочка делает сразу две вещи:
- `grep -i "error"` оставляет только строки с ошибками;
- `tail -n 2` берёт только две последние ошибки.

Именно так обычно строят быстрые команды для разбора логов.
