## Урок 13 пройден!

Теперь вы умеете:

- собирать архивы через `tar`;
- смотреть содержимое через `tar -tf`;
- распаковывать архивы через `tar -xf`;
- сжимать отдельные файлы через `gzip`;
- создавать backup-архивы в `tar.gz`;
- делать более плотные snapshot-архивы через `tar.xz`;
- восстанавливать один файл без полной распаковки.

Это очень практичная база для backup-задач, сборки артефактов и переноса конфигов между серверами.
