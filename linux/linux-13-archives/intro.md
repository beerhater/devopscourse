## Архивы и сжатие в Linux

Для DevOps-инженера архивы нужны постоянно:

- собрать backup конфигов перед изменением;
- сохранить артефакты сборки;
- передать лог-файлы коллегам;
- быстро восстановить один файл после неудачного деплоя.

В этом уроке вы разберёте:

- `tar` для упаковки каталога;
- `tar -tf` для просмотра содержимого;
- `tar -xf` для распаковки;
- `gzip` и `gunzip` для сжатия одного файла;
- `tar -czf` и `tar -cJf` для сжатых архивов;
- выборочное восстановление одного файла из backup.

Идём от простого архива к более реалистичному backup-сценарию.
