## Урок 14 пройден!

Теперь у вас есть сильный базовый набор для работы с текстом:

- `cut` для колонок;
- `tr` для простой нормализации;
- `sed` для замен;
- `awk` для выборки и вычислений;
- `xargs` для массовой передачи аргументов.

Эти инструменты очень хорошо дополняют `grep`, `find`, pipes и bash-скрипты.
