## Текстовая обработка в Linux

В DevOps много работы крутится вокруг текста:

- CSV и TSV-отчёты;
- `.env` и `.conf` файлы;
- списки серверов;
- инвентори, логи и артефакты пайплайнов.

В этом уроке вы на практике используете:

- `cut` для колонок;
- `tr` для замены символов и нормализации;
- `sed` для массовых замен;
- `awk` для выборки и простых вычислений;
- `xargs` для передачи списка файлов в команды.

Это те утилиты, которые помогают быстро решать задачи без открытия редактора и без лишних скриптов.
