## Нормализуем текст через tr

`tr` хорошо подходит для простых преобразований:

- заменить один символ на другой;
- разбить строку по разделителю;
- перевести буквы в нижний регистр.

Приведите список хостов к удобному виду: по одному имени в строке и в нижнем регистре.

```bash
tr ',' '\n' < /opt/text-lab/hosts.raw | tr 'A-Z' 'a-z' > /root/hosts_normalized.txt
cat /root/hosts_normalized.txt
```{{execute}}

Так часто готовят входные данные для циклов, скриптов и массовых проверок.
