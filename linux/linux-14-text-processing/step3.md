## Меняем строки через sed

`sed` полезен, когда нужно быстро заменить значения в конфиге без ручного редактирования.

Сделайте из dev-конфига prod-версию:

```bash
sed \
  -e 's/^APP_ENV=dev$/APP_ENV=prod/' \
  -e 's/^APP_PORT=3000$/APP_PORT=8080/' \
  /opt/text-lab/app.env > /root/app-prod.env

cat /root/app-prod.env
```{{execute}}

Это очень похоже на реальные задачи:

- подменить порт;
- переключить окружение;
- быстро подготовить конфиг под другой стенд.
