## Практика — собираем отчёт по prod-сервисам

Теперь объединим несколько инструментов и соберём мини-отчёт по продовым сервисам.

Нужно оставить только строки с `prod`, а затем превратить CSV в более читаемый вид:

```bash
awk -F, 'NR > 1 && $2 == "prod" {print $1 "," $3}' /opt/text-lab/deployments.csv | sed 's/,/ -> /' > /root/prod_images_report.txt
cat /root/prod_images_report.txt
```{{execute}}

Такой подход постоянно встречается в эксплуатации:

- собрать список образов для релиза;
- быстро проверить, что уходит в production;
- подготовить короткую сводку для команды.
