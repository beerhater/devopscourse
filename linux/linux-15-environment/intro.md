## Переменные окружения и PATH

Эта тема постоянно встречается в DevOps:

- `.env` файлы для приложений;
- переменные для CI/CD;
- `PATH` для собственных утилит;
- shell profile для постоянных настроек;
- разница между временной переменной и постоянной настройкой.

В этом уроке вы разберёте:

- `export`;
- переменную для одной команды;
- как скрипт читает env;
- как работает `PATH`;
- чем полезны `which` и `type`;
- как использовать `.bashrc` и `source`.

Это база для Docker, Kubernetes manifests, CI-пайплайнов и обычной работы в терминале.
