## Передаём переменную только одной команде

Иногда переменная нужна только для одного запуска.
Это очень удобно для одноразовых проверок и локальных тестов.

```bash
APP_ENV=prod bash -c 'echo $APP_ENV' > /root/one_run_env.txt
cat /root/one_run_env.txt
```{{execute}}

После выполнения команды переменная не становится глобальной для всей сессии.

Так часто запускают:

- временные проверки;
- разовые сборки;
- локальные dry-run команды.
