## Урок 16 пройден!

Теперь вы умеете:

- быстро собирать системную сводку;
- смотреть память через `free`;
- проверять место и inode через `df`;
- искать тяжёлые каталоги через `du`;
- смотреть устройства через `lsblk`;
- разбираться с mount points;
- находить крупные файлы через `find`.

Это очень практичный набор для первичной диагностики Linux-сервера.
