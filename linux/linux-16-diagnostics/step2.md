## Проверяем память

Память — одна из первых вещей, которые нужно проверить при деградации сервиса.

Снимите короткий отчёт по памяти:

```bash
free -h > /root/memory_report.txt
free -m > /root/memory_report_mb.txt

cat /root/memory_report.txt
```{{execute}}

`free -h` удобно читать человеку, а `free -m` даёт значения в мегабайтах для более точного сравнения.
