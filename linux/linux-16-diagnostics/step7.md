## Практика — находим самый тяжёлый артефакт

Теперь соберём маленький итоговый one-liner: найдём самый тяжёлый файл в учебном каталоге.

```bash
find /opt/diagnostics-lab -type f -printf '%s %p\n' | sort -n | tail -n 1 > /root/largest_file.txt
cat /root/largest_file.txt
```{{execute}}

Это уже очень похоже на реальную мини-диагностику:

- собрать размеры;
- отсортировать;
- взять самый тяжёлый объект;
- быстро понять, что именно забирает место.
