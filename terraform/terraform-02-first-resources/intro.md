# Terraform. Урок 2 — Первые ресурсы

В первом уроке мы создали минимальный `.tf`-файл и запустили workflow.
Теперь идём глубже: разбираем провайдер `local` во всех деталях,
учимся связывать ресурсы между собой и управлять зависимостями.

```
Что нового:
  local_file              <- создание файлов с настройкой прав
  local_sensitive_file    <- файлы с секретами (не светятся в логах)
  Ссылки на атрибуты      <- resource.type.name.attribute
  data sources            <- читать уже существующие файлы
  random_*                <- случайные значения: id, строки, пароли
  depends_on              <- явные зависимости между ресурсами
  lifecycle               <- управление жизненным циклом ресурса
```

> Начнём: `terraform version`{{execute}}
